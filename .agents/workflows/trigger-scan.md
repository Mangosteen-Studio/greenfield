# Trigger Scan Workflow

## Steps

1. Read the guardrails:
   - `guardrails/VOICE.md`
   - `guardrails/VERIFICATION.md`

2. Open `skills/trigger-radar/SKILL.md` — follow it step by step.

3. Collect: company name, industry, what you're selling.

4. Run the Trigger Radar interrogation and research actions.

5. Output the Trigger Radar report with Why Now Score.

6. Run the Stage 0.5 checkpoint from `guardrails/QUALITY_GATES.md`.

7. Based on the Why Now Score, recommend next steps:
   - 8-10: Move now → offer full Greenfield or warm path mapping
   - 5-7: Build slowly → offer trigger monitoring setup
   - <5: Park → suggest setting alerts and revisiting
