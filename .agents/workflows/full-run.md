# Full Greenfield Run

## Steps

1. Read and internalize the guardrails:
   - `guardrails/QUALITY_GATES.md` — stage checkpoints to enforce
   - `guardrails/VOICE.md` — output quality rules
   - `guardrails/VERIFICATION.md` — verification discipline

2. Open `GREENFIELD.md` — this is the canonical workflow.

3. Follow `GREENFIELD.md` stage by stage, in order:
   - Declare mode and deliver the opening orientation + research handoff
   - Stage 0 (Account Identity Disambiguation)
   - Stage 0.5 (Trigger Radar)
   - LinkedIn CSV decision
   - Pre-Stage (company context qualifier)
   - Stage 1 (Internal Intelligence)
   - Stage 2 (Company Website & Product)
   - Stage 3 (Hiring Signals)
   - Stage 4 (Market & Financial)
   - Stage 5 (Leadership Deep Dive)
   - Stage 5.5 (Executive Narrative & Access)
   - Stage 6 (Warm Path Activation)
   - Stage 7 (Competitive Intelligence)
   - Stage 8 (Industry Language)
   - Stage 9 (Persona Messaging)
   - Final Output (Account Brief)

4. At every stage transition, run the relevant checkpoint from `QUALITY_GATES.md`. Do not skip checkpoints.

5. Build the Account Brief live as the AE answers. Do not wait until the end.

6. After the Account Brief is complete, run the Account Brief Final Validation from `QUALITY_GATES.md`.

7. Output the human Completion Status block and append the JSON sidecar.

8. Offer to run `/account-review` for a quality audit of the completed Account Brief.
